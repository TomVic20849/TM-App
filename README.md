# Couple Cycle App

This is a couple-focused app inspired by apps like Flo, created as a free alternative.
The goal is to allow both partners to have access to useful information about the
menstrual cycle, promoting understanding and communication within the relationship.

## Project Goals
- Learn and practice mobile development using Expo and React Native
- Help my girlfriend learn how to code
- Build a real project that can be used as part of a portfolio

## Tech Stack
- Expo
- React Native
- TypeScript
